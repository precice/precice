* Deprecated Fortran functions `precicef_set_vertex_`, `precicef_set_edge_`, `precicef_set_triangle_`, `precicef_set_quad_`, and `precicef_set_tetrahedron`. Use the new `precicef_set_mesh_*` variants instead.
